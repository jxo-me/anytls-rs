| 症状 | 可能原因 | 排查步骤 |
|------|----------|----------|
| `authentication failed` | 密码不一致 | 确认 `anytls-server -p` 与 sing-box `password` 一致；检查是否有额外空格/换行 |

| `connection refused` | 服务端未监听或被防火墙阻断 | 确认 `anytls-server` 监听地址，使用 `lsof -i :8443` 检查端口；排除防火墙/安全组 |
| UDP 请求无响应 | 路由未放行 UDP 或 sing-box 配置遗漏 | 检查 sing-box `route.rules` 是否覆盖 `udp`，确认目标服务器允许 UDP |
| 长连接频繁重建 | 会话池参数未对齐 | 关注 `RUST_LOG` 输出；后续补丁提供 `idle_session_keepalive` 等配置入口 |
| 观察不到详细日志 | logging 未开启 | 使用 `RUST_LOG=info,anytls=debug`；确认 `tracing-subscriber` 初始化成功 |
